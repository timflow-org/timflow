Area-sinks
==========

1. :class:`~timml.circareasink.CircAreaSink` is a circular area-sink for modeling uniform
   infiltration over a circular area.

2. :class:`~timml.inhomogeneity.PolygonInhomMaq` or
   :class:`~timml.inhomogeneity.PolygonInhom3D` can specify recharge inside a
   polygonal area.
